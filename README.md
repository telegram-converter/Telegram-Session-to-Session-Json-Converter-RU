# STSJ Converter – Telegram **SESSION → SESSION+JSON**  
Небольшой инструмент, превращающий файлы сессий Telegram в формат JSON, совместимый со средами вроде Telegram Prime.

---

---

## ⚙️ Что делает конвертер

| Функция | Описание |
| --------|----------|
| **SESSION → SESSION+JSON** | Переводит нативные файлы сессий Telegram в JSON‑формат, совместимый с Telegram Prime и другими клиентами. |
| **Поддержка Pyrogram** | Принимает также файлы сессий из Pyrogram. |
| **Конвертация без подключения** | Возможность преобразовать SESSION+JSON в TDATA без онлайн‑соединения с аккаунтом. |
| **Совместимость со старым лаером** | Опция сохранения совместимости со старой версией Telegram лаера или запись как новый слой. |
| **Повторная авторизация** | Создаёт новую сессию при необходимости. |
| **Очистка сессий** | Опционально отключает все остальные активные сессии аккаунта во время конвертации. |
| **Настраиваемые попытки / задержки** | Устанавливает количество попыток подключения и паузу между ними. |
| **Многопоточность** | Позволяет запускать несколько преобразований параллельно, ускоряя массовую работу. |
| **Точная настройка JSON** | Изменяемые поля устройства, языка и прочее перед записью – ключ к избежанию блокировок. |
| **Сортировка** | Валидные сессии → «valid», недействительные → «invalid». |
| **Логи** | Полный вывод в консоль о ходе выполнения. |
| **Резервное копирование** | Оригинальные файлы сохраняются перед изменениями. |
